### 🚀 快速开始指南

1.  **找到主程序**：在文件夹中找到名为 **智能日程.exe** 的图标（通常是一个小机器人或蓝色方块图标）。
2.  **双击运行**：直接双击该文件即可打开软件（**无需安装 Python 环境**，已打包好）。
3.  **开始使用**：软件会自动在后台生成数据文件，你只需专注于添加任务即可。

> **💡 核心提示**：
> 软件运行后，会自动在**同一个文件夹**下生成一个名为 `smart\_tasks.json` 的文件。**请务必不要删除或修改这个文件**，它是你所有日程数据的“仓库”，删除会导致数据永久丢失。

### 📝 详细功能说明

1.  **添加任务**：在输入框中输入内容，点击“添加”按钮（或按回车键）。
2.  **标记完成**：点击任务前的方框，任务会自动变灰并移至列表底部。
3.  **删除任务**：选中任务后，点击右侧的“删除”按钮。
4.  **切换日期**：点击顶部的日历图标，可以查看或修改过去的任务。
5.  **数据统计**：左侧的日历视图中，绿色格子代表“已完成”，蓝色格子代表“已打卡”。

### 🔐 数据备份与安全

-   **备份方法**：如果你想更换电脑或防止数据丢失，**直接复制整个文件夹**（包含 `.exe` 和 `.json` 文件）到 U 盘或云盘即可。
-   **数据恢复**：将备份的文件夹复制到新电脑的任意位置，双击运行 `.exe` 即可继续使用。

### ❓ 常见问题 (FAQ)

**Q: 双击 智能日程.exe 没反应，软件打不开怎么办？**

A: 这通常是因为电脑的“杀毒软件”拦截了程序，或者系统没有管理员权限。
-   **解决方法**：右键点击 智能日程.exe，选择“以管理员身份运行”。
-   如果还是不行，请检查杀毒软件的“隔离区”，将其恢复并添加信任。

**Q: 为什么我的杀毒软件提示这个文件有病毒？**

A: 这是**误报**。因为软件是用 Python 打包的，很多杀毒软件会对这类打包文件产生误判。软件本身是安全的，你可以放心运行。

**Q: 可以在手机上使用吗？**

A: 不可以。本软件是为 Windows 电脑设计的桌面程序，不支持手机、平板或 Mac 电脑。
