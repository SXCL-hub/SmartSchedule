## 📥 下载与使用

### 方式一：直接运行 (推荐普通用户)
如果你不想折腾代码，可以直接下载编译好的程序：
1.  进入 `Releases` 页面。
2.  下载最新的 `智能日程.exe`。
3.  双击即可运行。
4.  保存好.json文件，它用于数据存储。


### 方式二：从源码运行 (推荐开发者)
如果你喜欢折腾代码：
1.  安装 Python 3.8 或更高版本。
2.  克隆本仓库：`git clone [你的仓库地址]`
3.  安装依赖：`pip install -r requirements.txt`
4.  运行：`python Smart_Schedule.py`
